# Automation Hub — Kingdom Control Center

Central automation dashboard for **The Royal Kissi Kingdom** digital ecosystem.

## Features

- **Free Account Manager** — Track usage across 14+ free-tier accounts (email, AI, hosting)
- **Cron Jobs** — Manage scheduled tasks across all Kingdom apps
- **Data Scanners** — Web scrapers for Sotheby's, county records, public data
- **GPU Orchestrator** — Manage cloud GPU resources for AI avatar processing
- **Security Center** — CRYSTALS-Kyber post-quantum security + audit
- **System Logs** — Centralized logging across all services
- **Deployment Monitor** — Track all 14 Kingdom apps

## Pages (10)

Dashboard, Free Accounts, Cron Jobs, Scanners, Security, GPU Orchestrator, System Logs, Deployments, Settings, Admin

## Tech Stack

- **Frontend:** React + Vite + Tailwind CSS
- **Hosting:** update.kckngdm.com (Vercel)

## Setup

```bash
npm install
npm run dev
```
